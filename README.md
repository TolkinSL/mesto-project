# Проект: Место

## Описание проекта

3-й проект на курсе Веб-разработчик плюс Яндекс Практикум.

Используемые технологии:

* Гридбокс-верстка
* Флексбокс-верстка
* Адаптивная-верстка
* Формы
* Методология БЭМ
* Файловая структура Nested

Ссылка [на проект](https://tolkinsl.github.io/mesto-project/) в Github Pages
